# 快乐小严腰痛处方生成器 - 启动说明

## 快速启动

### Windows 用户

**方法1：双击运行（推荐）**
- 双击 `start_server.bat` 文件
- 会自动启动本地服务器
- 在浏览器中访问 `http://localhost:8000`

**方法2：PowerShell运行**
- 右键点击 `start_server.ps1`
- 选择"使用PowerShell运行"
- 或在PowerShell中执行：`.\start_server.ps1`

### Linux/Mac 用户

**方法1：命令行运行**
```bash
# 添加执行权限（首次运行）
chmod +x start_server.sh

# 运行脚本
./start_server.sh
```

**方法2：直接运行**
```bash
bash start_server.sh
```

## 使用步骤

1. **运行启动脚本**
   - Windows: 双击 `start_server.bat`
   - Linux/Mac: 运行 `./start_server.sh`

2. **等待服务器启动**
   - 看到 "服务器地址: http://localhost:8000" 表示启动成功

3. **打开浏览器**
   - 访问 `http://localhost:8000`
   - 打开 `index.html` 页面

4. **使用工具**
   - 选择图片
   - 调整次数
   - 生成PDF

5. **停止服务器**
   - 在命令行窗口按 `Ctrl+C`

## 常见问题

### Q: 提示"未检测到Python"
**A:** 需要先安装Python 3.x
- Windows: 访问 https://www.python.org/downloads/
- Linux: `sudo apt-get install python3` (Ubuntu/Debian)
- Mac: `brew install python3`

### Q: 提示"端口8000已被占用"
**A:** 可以修改脚本使用其他端口
- 将脚本中的 `8000` 改为其他端口（如 `8080`）
- 然后访问 `http://localhost:8080`

### Q: 服务器启动后无法访问
**A:** 检查以下几点
1. 确保防火墙允许访问
2. 确保浏览器访问的是 `http://localhost:8000`（不是 `file://`）
3. 检查命令行窗口是否显示错误信息

## 注意事项

⚠️ **重要**：必须使用本地服务器运行，不能直接双击打开HTML文件！

- ❌ 错误方式：直接双击 `index.html`（file://协议）
- ✅ 正确方式：运行启动脚本，然后访问 `http://localhost:8000`
